Other version control systems are gitlab etc.

Useful git command lines:
- git --version
- git config --list
- git config user.name # to know the current user name
- git config --global user.name "new user name"
- git config user.email # to know the current user name
- git config --global user.email "new user email"
- git init # this command is to initiate git in the current working directory
- git add <filename> # does change control/track
- git status
- git diff
- git log
